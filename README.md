## Como clonar um repositório 

1. Passo a passo: 

```bash
# Navegando até o disco C
cd /c

# Criando uma pasta
mkdir minha-pasta

# Entrando na pasta 
cd minha-pasta 

# Clonando repositório
git clone <link-repositório>

# Acessando o repositório
cd <path-repositorio>
```
